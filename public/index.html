Timestamp Microservice

This application returns a given date in two forms: the natural language date (e.g., January 1, 2017) and the unix timestamp (i.e., the number of seconds since January 1, 1970). If the entered value is not a date, the service returns null.

Instructions:

Enter a date (either unix or natural language) as a parameter in the browser address bar.

For example, try one of the following inputs:

  • https://obscure-sierra-15573.herokuapp.com/1450137600
  • https://obscure-sierra-15573.herokuapp.com/December 15, 2015

Example output:

  • { "unix": 1450137600, "natural": "December 15, 2015" }

Enjoy!
